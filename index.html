The link to the page

  • The base of hypertext. The best friend of the search engine;
  • Navigate the user to a new page or view;
  • Convenient and user-friendly;
  • The main thing is address.
			<a href="http://example.com/">
			    Link
			</a>

The link to the fragment

			
				<a href="#section-3">
					Section №3
				</a>
			
			
				<section id="section-3">
					<h2>Section №3</h2>
				</section>

Placeholder hyperlink

			<ul>
				<li><a href="/">Home</a></li>
				<li><a href="/news">News</a></li>
				<li><a>Examples</a></li>
				<li><a href="/legal">Legal</a></li>
			</ul>

Forget about those span element replacements used to style links targeting the current page. Remove the href attribute from the link instead.

The four links states are:

			a:hover { color: blue }
			a:focus { outline: 1px solid blue }
			a:active { color: red }
			a:visited { color: violet }

We shouldn't use only :hover.

Кнопки 90-х

			<form>
				<input type="submit" value="Button">
				<input type="button" value="Button">
				<input type="image" src="button.png">
			</form>

<input type="image"> передаёт координаты клика гетом.

The button element represents a button. The element is a button.

W3C Working Group Note

Кнопка «кнопка»

			<button type="button">Кнопка</button>
			<button type="submit">Кнопка</button>
			<button type="reset">Кнопка</button>

Универсальный элемент для использования вне форм.

Просто <button> даёт тип submit.

Состояния кнопок

			button:hover { background: darker }
			button:focus { outline: 1px solid blue }
			button:active { background: inverted }

Состояние :focus показывает текущее положение.
